How to add and edit the divider</p>
<p>The Divider Block can be used to insert a simple line, which can serve to divide the content.</p>
<p>How to add and edit the divider
You can edit The Divider Block by clicking it. Here, you can toggle the Custom padding option and change the size of the space around the divider, similar to The Space Block.</p>
